# 🌿 Nutritional Data Engine: Health Analytics Prototype

[![Live Demo](https://img.shields.io/badge/Live%20Demo-Test%20the%20Lab-success?style=for-the-badge&logo=vercel)](https://nutritional-data-engine.vercel.app/)

## 💡 The Concept: Data Analytics Beyond Business
Data analysis isn't limited to e-commerce conversion rates or sports metrics; it can optimize our daily routines. This project is an interactive **Nutritional Prototype** built to simulate the physiological impact of natural extracts compared to commercial energy drinks.

Instead of traditional KPIs, this interactive dashboard acts as a "Custom Blend Simulator." It uses a JavaScript-based engine to analyze the real-time impact of adjusting active ingredients on Focus, Immunity, and Metabolism.

## 🧪 The Formulation Engine
The simulation models the efficacy of four core natural extracts:
* **Cinnamon:** Evaluated for its impact on metabolic rates.
* **Ginger:** Modeled for its immunity-boosting properties.
* **Fenugreek:** Analyzed for blood sugar regulation and sustained endurance.
* **Turmeric:** Scored for its antioxidant payload and cognitive focus extension.

## ✨ Key Features
* **Interactive Sliders:** Users act as "formulators," adjusting milligram (mg) dosages of each extract.
* **Synthetic Simulation Engine:** A custom Vanilla JS engine instantly recalculates efficacy scores based on input ratios.
* **Dynamic Visualizations:** Features a real-time Radar Chart and Bar Chart (via Chart.js) to visually compare the custom natural blend against a standard benchmark (a high-sugar, synthetic caffeine energy drink).
* **Responsive UI:** Clean, modern "Lab Mode" interface built with Tailwind CSS.

## 🛠️ Tech Stack
* **HTML5 & Vanilla JS:** For structure and the simulation logic engine.
* **Tailwind CSS:** For rapid, modern UI styling.
* **Chart.js:** For dynamic, animated data visualization.
